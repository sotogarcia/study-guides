## LibreOffice Writer

    - Procesador de textos
    - WYSIWYG: What You See Is What You Get


### Acciones comunes

    - Nuevo		Archivo / Nuevo				CTRL + U  ; "CTRL + N"

    - Abrir		Archivo / Abrir				CTRL + A  ; CTRL + O

    - Cerrar		Archivo / Cerrar			CTRL + F4 ; CTRL + W

    - Guardar		Archivo / Guardar			CTRL + G
	· La primera vez respondo: ubicación, nombre y tipo
	· Las sucesivas usa los mismos valores

    - Guardar como	Archivo / Guardar como			CTRL + MAYÚS + S
	· Todas las veces pregunta: ubicación, nombre y tipo
	· Concluida la acción me deja con la copia

    - Guardar como copia	Archivo / Guardar como copia
	· Todas las veces pregunta: ubicación, nombre y tipo
	· Concluida la acción me deja con el ORIGINAL

    - Imprimir		Archivo / Imprimir			CTRL + P

    - Vista previa	Archivo / Previsualización de impresión	CTRL + MAYÚS + O



### Modos de escritura

    - Inserción		: el texto de la derecha es empujado para dar cabida al nuevo
    - Sobrescribir	: el texto de la derecha es reemplazado carácter a carácter

### Borrado

    + Hacia la izquierda

	- Retroceso		: elimina carácter a carácter hacia la izquierda
	- CTRL + Retroceso	: elimina una plabra cada vez hacia la izquierda

    + Hacia la derecha

	- SUPR			: elimina carácter a carácter hacia la derecha
	- CTRL + SUPR		: elimina una palabra cada vez hacia la derecha


### Selecciones

    + F8

	Una vez		: una palabra
	Dos veces	: una frase (hasta el siguiente punto o enter)
	Tres veces	: un párrafo (hasta el siguiente enter)

    + Clics de ratón

	Dos clics	: un palabra
	Tres clics	: una frase
	Cuatro clics	: un párrafo

    + CTRL + E
	
	Selecciona "todo" lo que hay en el área en la cual está el cursor
	Sucesivas pulsaciones de esta combinación extienden la selección a un área mayor

## Barra de estado

- Icono Documento modificado
- Nº de página sobre el total de páginas
- Nº de palabras sobre el total de palabras
	· Recuento de la selección, si existe
	· Recuento para todo el documento, si no se ha seleccionado nada
- Estilo aplicado a la página en la cual está el cursor
- Idioma de corrección
- Modo de escritura:
	Sobrescribir
	Insertar (aparece vacío)
- Firmas
- Información diversa
	· Nombre de la sección
	· Nombre de tabla y celda	tabla:C1
	· Nivel en la lista
	· Nivel de título
	· Vistas y zoom
